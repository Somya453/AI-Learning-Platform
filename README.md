## 🚀 About the Project

**AI LearnPro** is an interactive AI-based learning platform designed to make studying fun, productive, and engaging.  
It combines **AI assistance, gamified quests, study rooms, and productivity tools** in one place.

Students can:
- 🧠 Learn AI concepts through quests
- 📚 Join or create study rooms
- ⏱️ Use Pomodoro focus timer
- 🎵 Listen to focus music
- 📝 Manage daily tasks
- 📊 Track progress & performance

## ✨ Features
